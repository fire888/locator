
# VEHICLE LOCATOR 

About
------------   
Online browser game on *nodejs* and *threejs*.  
  
  
![pic](https://raw.githubusercontent.com/fire888/locator/master/jsScene/screenshot.jpg)  

Gameplay
------------ 
||| 
| -------------------- | ------------------:|
| **_Vehicle:_**       |                    |
| move forward         | "key up"           |
| move backward        | "key down"         |
| turn left            | "key left"         |
| turn right           | "key right"        |
| 	                   |                    |
|turn gun left         | "A"                |  
|turn gun right        | "D"                |  
| **_Player:_**        |                    |
|look around           | "mouse move"       |  
|move forward          | "key up"           |
|move backward         | "key down"         |
