About
------------   
Online browser game on *nodejs* and *threejs*.  
  
  
![pic](https://raw.githubusercontent.com/fire888/locator/master/www/app/styles/screenshot.jpg)  

Gameplay
------------ 
||| 
| -------------------- | ------------------:|
| **_Vehicle:_**       |                    |
| move forward         | "key up"           |
| move backward        | "key down"         |
| turn left            | "key left"         |
| turn right           | "key right"        |
| 	                   |                    |
|turn gun left         | "key A"            |  
|turn gun right        | "key D"            |  
| **_Player:_**        |                    |  
|move forward          | "key up"           |
|move backward         | "key down"         |
|move left             | "key A"            |
|move right            | "key D"            |
|turn left             | "key left"         |  
|turn right            | "key right"        |
