
# VEHICLE LOCATOR 

Online browser game on *nodejs* and *threejs*  
[Play](http://js.otrisovano.ru/tests/180316Locator/01) and check bugs :)


Gameplay
------------ 
||| 
| -------------------- | ------------------:|
| **_Vehicle:_**       |                    |
| move forward         | "key up"           |
| move backward        | "key down"         |
| turn left            | "key left"         |
| turn right           | "key right"        |
| 	                   |                    |
|turn gun left         | "A"                |  
|turn gun right        | "D"                |  
| **_Player:_**        |                    |
|look around           | "mouse move"       |  
|move forward          | "key up"           |
|move backward         | "key down"         |
